How to play

Fill every empty cell so each row, column and 2×3 box contains the numbers 1 to 6 with no repeats — conflicts light up red.

  • Tap / ArrowsSelect a cell
  • Tap pad / 1–6Place a number
  • Erase / 0Clear a cell
  • SpaceNext empty cell
  • PPause
  • RNew puzzle
