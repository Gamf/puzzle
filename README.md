# Puzzle — a Gamf genre

Published at <https://gamf.github.io/puzzle/>.

- `games.json` — the list of games in this genre (the hub reads this).
- `games/<slug>/` — one self-contained folder per game.
- `index.html` — this genre's own landing page.

Add a game: copy a template from `../templates/`, drop it in `games/`, add a row
to `games.json`. See `../docs/ADD-GAME.md`.
